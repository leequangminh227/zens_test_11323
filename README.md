## Description

<h3>3 parts: </h3>
    <div>zens_test_fe: website for joke display and react</div>
    <div>zens_test_BE: api and unittest written by Nestjs framwork and swagger</div>
    <div>algorithm: algorithm solving and test cases in python</div>




## algorithm
```bash
$cd algorithm

#running sample code
$py algorithm.py

#running testcase
$py test.py
```
## FE

```bash
$cd zens_test_fe

#running the website 
$npm start 

```
## BE

```bash
$cd zens_test_BE

#running the app 
$npm run start

#API doc with Swagger 

go to http://localhost:3000/api

#unit test
$ npm run test:watch

```
